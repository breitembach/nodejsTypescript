## NODEJS TYPESCRIPT

Project base Nodejs with Typescript;

### DEVELOPMENT
`> npm run dev`

## PRODUCTION
`> npm run build`

`> node src/server.js`

### USAGE
`> npm install -g typescript`

`> npm install -g eslint`

`> npm install`

### USE VSCODE?
* install plugin `eslint`

#### vscode settings
```
{
"eslint.autoFixOnSave": true,
    "eslint.validate": [
        "javascript",
        {"language": "typescript", "autoFix": true},
    ]
}

```
