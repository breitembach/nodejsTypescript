## NODEJS TYPESCRIPT

Project base Nodejs with Typescript;

### DEVELOPMENT
`> npm run dev`
#### OPCIONAL
`install docker and docker compose for RUN`

### PRODUCTION
`> npm run build`

`> node src/server.js`

## USING DOCKER
`> docker-compose up`


### USAGE
`> npm install -g typescript`

`> npm install -g eslint`

`> npm install`

### USE VSCODE?
* install plugin `eslint`

#### vscode settings
```
{
"eslint.autoFixOnSave": true,
    "eslint.validate": [
        "javascript",
        {"language": "typescript", "autoFix": true},
    ]
}

```
